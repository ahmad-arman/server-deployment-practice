## Setup
1. `(.env)` file 
```
PORT = 3000

```
2. Install the following dependencies
```
npm init -y 

npm i -D jest supertest eslint

npm i dotenv express
```
## Running the app 
1. clone the repo.
2. Enter the command `npm start`
3. Use endpoints :/error 
 


4. Test .  
* There's 1 test files . server.test.js
* In terminal run :

```
npm test
```
